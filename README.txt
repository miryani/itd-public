
---
How-to Hugo? 

1- Install Hugo:

brew install hugo
hugo version


---

Clone the git project

1- install git 
2- create a folder like: mkdir itd-project
3- cd itd-project
4- git init
5- git add https://github.com/miryani/itd-public.git 
6- launch the web server : hugo server -D
7- browse the website: http://localhost:1313/
